1. [Flow test]

Line-up

  1. X-mas Tree
  2. eruption manifold
  3. rock-catcher
  4. Choke-manifold
  5. separator
  6. Sealine

Equipments

  1. X-mas Tree
    • Swap is open
    • M/V is open
    • W/V is open
    • DHSV is open
  2. eruption manifold
    • lined up from the well to the rock-catcher
  3. rock-catcher
    • we can flow through the filter/ strainer or we can bypass it
  4. Choke-manifold
    • we can use adjustsble side (if we are using for a short time) or fixed side (if we are using for a short time) the test takes four hours for each choke size
  5. separator
      we can bypass separator or we can flow through it
    • Oil Line:-
      • oil line is open
      • Setting LCV
      • Checking flow meter
    • Oil Line:-
      • Gas line is open
      • Setting PCV
      • Checking separator pressure
      • once we runing the orifice plate and closing the equalizer of the differential sesnor (DPU) we monitor Diff. Press.
  6. Sealine
    • sealine diverter valve is open
    • sealine valve is open

Operation

  1. Samples:-
    • BS&W (Water Cut)
    • API & Temp.
    • H2S & CO2
    • shrinkage
    • Gas SG
  2. Gas Line
    • Setting PCV
    • Monitoring Separator pressure
    • After Stabilizing the level we can run the orifice plate to start flow test
  3. Oil Line
    • Setting the LCV
    • Monitoring Separator Level
    • Monitoring Flow meter
  4. Monitoring
    • WHP
    • WHT
    • DSP
    • SEP.P.
    • SEP.T
    • SLP
    • SEP. Level
    • Annulus (A)
    • Diff. Press.
    • Flow meter
    • Delta Press. for the rock-catcher

Notes

  • Set the adjustable choke size before opening the choke
  • increasing the adjustable choke size gradually
  • always open downstream valve before upstream one
  • opening upstream chokemanifold valve slowly to avoid hydration and going down WHP suddenly
