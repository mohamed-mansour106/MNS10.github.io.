1. API & K-Factor Correction

Step 1: Correcting API to 60°F

$$ \rho_{obs} = \frac{141.5}{API_{obs} + 131.5} \times 999.012 $$

The corrected density ($\rho_{60}$) is solved using the temperature expansion coefficient ($\alpha$):

$$ VCF = e^{ [-\alpha \Delta T (1 + 0.8 \alpha \Delta T)] } $$

Step 2: K-Factor (VCF) for Line Temp

Adjusts the volume from meter/tank temperature to standard conditions:

$$ K = \frac{Volume_{60}}{Volume_{line}} $$
