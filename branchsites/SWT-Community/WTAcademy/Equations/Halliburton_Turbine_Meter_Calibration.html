Halliburton Turbine Meter Calibration

Calculate pulses per barrel and display values for MCII Analyzers.

Pulses per bbl

$$K_{bbl} = K_{gal} \times 42$$

TOP Display (bbls)

$$Disp_{top} = \frac{K_{bbl}}{100}$$

BOTTOM Display (Factor)

$$Factor = \frac{86,400}{K_{bbl}}$$

Based on $(1440 \times 60) / K_{bbl}$
